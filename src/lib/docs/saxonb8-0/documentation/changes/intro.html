Version 8.0

This page lists changes in Saxon 8.0 (issued 2004-06-01) relative to version 7.9.1. The list applies both to the open source Saxon-B 8.0 product and to Saxon-SA, the schema-aware product.

Some of the changes made in this release reflect Working Group decisions made subsequent to the publication of the November 2003 drafts. In most cases these decisions have been announced publicly on the public-qt-comments list.

For a change history prior to version 8.0, see the SourceForge site.

The changes in version 8.0 are listed under the following headings:

  • Defects cleared in Version 8.0

  • XSLT changes

  • XPath changes (also affecting XQuery)

  • XQuery changes

  • Functions and Operators

  • Schema processing

  • XPath API changes

  • Changes to extension functions

  • Changes to internal APIs

  • Implementation changes

  • Running Saxon as an Applet
